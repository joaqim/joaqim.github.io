Aug 08 2020 AIDungeon Import & Export Plugin

Grease­mon­key Plug­in

Warning:

The script might over­write ex­ist­ing en­tries (It re­al­ly shouldn’t but I haven’t test­ed it enough,) use with cau­tion.

Link to greasy­fork plug­in: AIDun­geon - Im­port and Ex­port World En­tries as Json

Example of Json of World Entries:

[
   {"key": "This is a key", "entry": "This is the World Entry"},

   {"key": "This is another key", "entry": "This is another World Entry"},

   {"key": "This is a key", "entry": "This is the World Entry"}
]


You can al­so add op­tion­al “show”: true/false ( if not as­signed, it de­fault to false ) to tog­gle hid­den world in­fo:

{"key": "key", "entry": "entry", "show": true }


Note

The scripts tries to em­u­late the us­er click­ing on “Add World In­fo” but­ton to fill the next en­try, this gets slow­er and slow­er the more en­tries you have. This has some­thing to do with AIDun­geon it­self, not the script.


Feed­back or sug­ges­tions are wel­come.
